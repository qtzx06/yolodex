# Subway Surfers Team Export

This package contains sanitized, team-ready YOLO training data.

## Contents
- `dataset/images/`: 30 frame images
- `dataset/labels/`: 30 YOLO label files (`class_id cx cy w h`)
- `classes.txt`: class id mapping order
- `preview/`: high-contrast bbox overlays for QA

## Classes
0 player
1 train
2 coins
3 powerup
4 obstacle
5 barrier

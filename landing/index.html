Key Innovation

Parallel Subagents

Frames are split into batches and labeled concurrently by N Codex agents, each in its own git worktree for filesystem isolation.

Ralph Loop

Autonomous iteration: collect → label → augment → train → eval → improve, until target accuracy is reached.

Codex Skills

Each pipeline stage is a standalone Codex skill with its own SKILL.md, independently runnable and composable.
